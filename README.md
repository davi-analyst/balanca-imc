# Calculadora de IMC

Calculadora web simples para calcular o Índice de Massa Corporal (IMC), desenvolvida com HTML, CSS (inline) e JavaScript.

## Sobre o IMC

O IMC (Índice de Massa Corporal) é uma medida internacional usada para calcular se uma pessoa está no peso ideal, abaixo ou acima do peso. A fórmula é amplamente recomendada pela **Organização Mundial da Saúde (OMS)** como uma das formas de avaliar a composição corporal.

## Como usar

Acesse a calculadora online: 
[Calculadora de IMC](https://davi-analyst.github.io/balanca-imc)

Basta inserir seu peso (em kg) e altura (em metros), separando decimais com ponto (ex: `65.7` ou `1.75`), e clicar em **Calcular IMC**.

## Tecnologias utilizadas

- HTML
- CSS (inline)
- JavaScript

Projeto desenvolvido para praticar manipulação de DOM com JavaScript, validação de entradas e cálculos matemáticos simples. Ideal para iniciantes que desejam aprender a integrar lógica e interface.
